About This Generator

            

                This implements a Linear Congruential Generator (LCG) based on the formula
                used in old C standard library implementations:
            

            

                Xn+1 = (1103515245 √ó Xn + 12345) mod 231
            

            

                This was the actual implementation used in older versions of glibc rand()
                and is known for producing poor quality randomness.
            


            
Known Weaknesses

            
    
                
  • Low-dimensional correlation: Sequential values are correlated in predictable patterns
    • 
                
  • Short period: The sequence repeats after 231 values
    • 
                
  • Spectral test failures: Points form hyperplanes in higher dimensions
    • 
                
  • Low-order bits: The least significant bits have very short periods
    • 
                
  • Predictability: The next value can be computed from previous values
    • 
            


            
Why Test With Poor RNGs?

            

                Testing with known-poor generators helps validate that statistical tests can detect real-world
                weaknesses. Many legacy systems still use LCGs, so understanding their failure modes is important.
